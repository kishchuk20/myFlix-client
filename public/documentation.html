Movie API Documentation

This API allows users to retrieve movie data and manage user accounts and favorite movies.

Method Endpoint Description Request Body Response Example
GET / Welcome message None 
"Welcome to my Movie API!"
GET /movies Returns all movies None 
[{ "Title": "...", "Year": "...", ... }]
GET /movies/:title Returns one movie by title None 
{ "Title": "...", "Genre": { "Name": "..." }, ... }
GET /movies/genre/:genreName Returns all movies matching genre None 
[{ "Title": "...", "Genre": { "Name": "..." }, ... }]
POST /users Create a new user
